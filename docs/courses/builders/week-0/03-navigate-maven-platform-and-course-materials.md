# Navigate Maven Platform and Course Materials

## Overview

This lesson shows you how to navigate the Maven platform, find course materials, access recordings, and understand what's included in the course.

---

## Navigating the Maven Platform

### Course Homepage

**Location:** https://maven.com/james-gray/claude

**What you'll find:**
- Course syllabus with all modules and lessons
- Live session schedule and Zoom links
- Session recordings
- Downloadable materials and resources
- Community discussion boards

### Module Structure

Each module contains:
- **Written lessons** - Read before live sessions (like this one!)
- **Live sessions** - Interactive teaching and implementation
- **Exercises** - Hands-on practice between sessions
- **Resources** - Supplementary materials, links, downloads

**How to navigate modules:**
1. Start with written lessons (pre-work)
2. Attend live sessions for instruction and Q&A
3. Complete exercises to apply what you learned
4. Review resources for deeper understanding

### Joining Live Sessions

**Before each session:**
1. Log into Maven 5 minutes early
2. Navigate to the current week's module
3. Click "Join Live Session" (Zoom link)
4. Test your audio/video before session starts

**Technical requirements:**
- Stable internet connection (wired recommended)
- Working webcam and microphone
- Zoom installed and updated
- Ability to share screen for troubleshooting

### Downloading Materials

**Where to find downloads:**
- Source materials stored on Google Drive as Google Slides
- Course resources available in the GitHub repository
- Each module may have links to both Google Drive and GitHub resources

**Recommended downloads:**
- Save all materials to a dedicated course folder on your machine
- Clone GitHub repos to your local environment
- Bookmark frequently referenced resources

**Note on slide decks:**
- Slide decks are available as Google Slides
- PDFs will be provided for students unable to access Google Drive links from corporate networks

## Course Materials & Resources

### What's Included

**Live instruction:**
- 10 live sessions (2 per week for 5 weeks)
- Real-time demos and walkthroughs
- Interactive Q&A and troubleshooting

**Recordings:**
- Lifetime access to all session recordings
- Ability to review at your own pace
- Downloadable for offline viewing

**Written materials:**
- Pre-session lessons and guides
- Step-by-step implementation documentation
- Supplementary reading and resources

**Code & templates:**
- GitHub repository with all code examples
- Notion templates for Agentic AI Registry
- Claude Skills starter files and examples

**Community access:**
- Private Slack workspace
- Peer collaboration and support
- Monthly Builder Forum (LinkedIn Group with ongoing live sessions)

**Coaching:**
- On-demand 1:1 sessions with James (15 or 30 minutes, as many as you need)
- Weekly office hours (Saturdays)
- Direct instructor support in Slack

### External Tools You'll Use

You'll work with tools outside of Maven. These are part of your AI Builder Stack:

- **Claude.ai** - Your primary AI workspace (requires paid subscription)
- **GitHub** - Version control for Skills and projects
- **Cursor or VS Code** - Code editor for building Skills
- **Claude Desktop** - Local Skills deployment and workflow execution
- **Claude Code** - Command-line agentic coding tool
- **Google Chrome** - Browser automation with Claude in Chrome extension
- **Notion** - Agentic AI Registry (optional but recommended)

See the "Identify Your AI Builder Stack" lesson for complete details on required tools.
