# Getting Started

Setup guides to get your AI builder environment ready. Follow these in order if you're new, or jump to specific guides as needed.

## Essential Setup

Start here for a complete AI builder environment:

| Guide | What it does |
|-------|--------------|
| [Claude Code Installation](claude-code-install.md) | Install the Claude Code CLI and extension |
| [Claude Projects](claude-projects-setup.md) | Set up Claude Projects for organized AI work |
| [Claude Personalization](claude-personalization-setup.md) | Configure Claude to understand your context |
| [Editor Setup](editor-setup.md) | Configure Cursor or VS Code |
| [Git Installation](git-install.md) | Install Git for version control |
| [GitHub Setup](github-setup.md) | Create and configure your GitHub account |

## Optional Tools

Additional tools depending on your workflow:

| Guide | What it does |
|-------|--------------|

## Recommended Order

If you're setting up everything from scratch:

1. **Git** - Version control foundation
2. **GitHub** - Where your code lives
3. **Editor** - Your development environment
4. **Claude Code** - The AI CLI
5. **Claude Projects** - Organize your AI work
6. **Claude Personalization** - Make Claude work better for you

## Prerequisites

- macOS or Windows (admin access required)
- Claude Pro, Max, or Team subscription
- Internet connection
